### 位与。A&B，A和B对应的二进制数位都为1时,结果才为1,其他情况为0. 
```
A     =  001101 // 13
B     =  100101 // 37
A & B =  000101 // 5
```

### 位或。A | B : A和B对应的二进制数位都为0时，结果才为0,其他情况为1.
```
A     =  001101 // 13
B     =  100101 // 37
A | B =  101101 // 45
```

### 左移操作。A << B：将A的二进制表示的每一位向左移B位， 左边超出的位截掉，右边不足的位补0。在取值范围内,移动一位相当于乘2.
```
A     =  001101 // 13
A << 1 = 011010 // 26
```
